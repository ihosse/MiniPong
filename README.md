# MiniPong
"Mini Pong" is a small game based on the classic Pong (1972) - Part of an exercise proposed in my Unity course (simple Physics and controls).

## Instructions:
- Play with a friend
- Hit the ball back
- Controls:
    Player 1: W and S
    Player 2: O and L
